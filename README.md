

Docker - Bytemark Symbiosis Server
======================================

Pull to docker:
```bash
sudo docker pull stilliard/bytemark-symbiosis
```

Or via git clone:
```bash
# Clone the repo
git clone https://github.com/stilliard/docker-bytemark-symbiosis.git
# Build the image
make build
# Run as a container:
make run
```
